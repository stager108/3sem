Tricky Mutex [tricky] - Бонусная
Рассмотрим следующую реализацию мьютекса:
class tricky_mutex {
public:
 tricky_mutex() : thread_count(0)
 {}
 void lock() {
 while (thread_count.fetch_add(1) > 0) {
 thread_count.fetch_sub(1);
 }
 }
 void unlock() {
 thread_count.fetch_sub(1);
 }
private:
 std::atomic<int> thread_count;
};
Гарантирует ли он взаимное исключение и свободу от взаимной
блокировки?
